# What is a templating language?

---

EJS is:

- Superset of HTML
- allows you to "mix" JS to create dynamic HTML

---

### Example

```html
<ol>
  <% friends.forEach((name) => { %>
  <li><%= name %></li>
  <% }); %>
</ol>
```

---

We **compile** a template to plain HTML.

---

```js
let template = `
<ol>
  <% friends.forEach((name) => { %>
    <li><%= name %></li>
  <% }); %>
</ol>
`;

let data = ['Anjula', 'Bouchard', 'Chad'];

ejs.render(str, data);
// => Rendered HTML string
```

```html
<ol>
  <li>Anjula</li>
  <li>Bouchard</li>
  <li>Chad</li>
</ol>
```
